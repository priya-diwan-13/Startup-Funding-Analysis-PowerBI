# Startup Funding Analysis Dashboard

## Project Overview

The Startup Funding Analysis Dashboard is an interactive Power BI project developed to analyze startup funding trends, revenue performance, return on investment (ROI), and business growth across different industries.

The dashboard helps users understand how funding impacts startup performance through interactive visualizations and KPI metrics.

---

## Objectives

- Analyze startup funding across industries
- Compare funding with generated revenue
- Measure Return on Investment (ROI)
- Identify high-performing industries and startups
- Provide business insights for investment decisions

### 2. Startup Performance Analysis

### 3. Startup Performance Insights

This page provides business insights.
